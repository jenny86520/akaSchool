本日教學導覽
  1. Reset 的多種復原方式
    1. 取消索引（git reset／git reset HEAD）
    2. 復原最新版本（git reset HEAD^／git reset HEAD~N）
    3. 復原 Merge、Reset、Revert（git reset ORIG_HEAD）
    4. Revert 常用參數（--patch／-p、--mixed、--hard）
  2. Checkout 與索引比較的復原方式
    1. 復原檔案／資料夾（git checkout -- <file>）
    2. 復原特定版本下的檔案／資料夾
  3. 復原版本（Revert）
  4. 重定基底（Rebase）
